Descripción del Proyecto
Este proyecto consiste en un simulador didáctico interactivo para niños, desarrollado con HTML, CSS y JavaScript, integrando manipulación del DOM y manejo de eventos.
El objetivo principal es crear una experiencia educativa dinámica donde el usuario interactúa con botones, recibe retroalimentación inmediata y avanza dentro del circuito de actividades.

En la Entrega N°1, desarrollé el simulador ALMA TREK, el cual estaba basado principalmente en lógica con prompt, alert y console.log.

En esta segunda entrega:
Se reemplazó la interacción por consola por interacción visual.
Se implementó manipulación del DOM.
Se trabajó con eventos reales del usuario.
Se creó una experiencia más dinámica y orientada a una aplicación web.
Esto demuestra la evolución del proyecto hacia una aplicación más completa e interactiva.
